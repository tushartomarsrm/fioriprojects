# SAP Fiori Projects

This repository contains a list of small SAP Fiori / SAPUI5 projects created for learning and practice purposes.  

---

## Projects

### 1. Northwind Orders Display app

A simple SAP Fiori application that displays orders from the Northwind OData service.

**Features**
- Orders displayed in card format
- Search orders by Customer Name
- Click on a card to view order details in a dialog
- Built using SAPUI5 with XML Views and MVC pattern

**Tech Used**
- SAPUI5
- JavaScript
- XML Views
- OData V2 (Northwind)

---

### 2. Product CRUD App

A simple SAP Fiori application that performs CRUD operations on products using an OData V2 service.

**Features**

* Display products in a responsive table
* Inline row-level edit and save functionality
* Create new products by duplicating existing ones
* Delete products directly from the table
* Automatic data binding with OData V2
* UI state management using JSONModel

**Tech Used**

* SAPUI5
* JavaScript
* XML Views
* OData V2

### 3. Full-Stack Authentication App (Fiori + CAPM)

A full-stack authentication application built using **SAP Fiori** and **SAP CAPM**, providing **Login, Signup, and Forgot Password** functionality.
The Fiori frontend and CAPM backend are connected using an **AppRouter** to overcome CORS issues and enable seamless communication.
After successful authentication, the user is redirected to the **Home page**.

---

## Features

* User Login
* User Signup
* Forgot Password flow
* Redirect to Home page after login
* Fiori and CAPM connected via AppRouter
* CORS issues handled using AppRouter

---

## Tech Used

* SAPUI5 (Fiori)
* SAP CAPM
* JavaScript
* XML Views
* OData
